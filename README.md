# Convenient Store Management System

A **desktop-based store management system** that allows efficient management of employees, products, suppliers, categories, and sales, with a built-in billing system.

---

## Features

### Dashboard
- Displays all key information about employees, products, suppliers, categories, and sales.
- Interactive tabs to navigate different sections.

### Employee / Product / Supplier Management
- Add, update, remove, and clear data.
- View existing records in a table format.
- Search functionality for easy record lookup.

### Categories Management
- Add and remove product categories.

### Sales Management
- View all bills generated by the store.
- Automatically stores all bill data.

### Billing Page
- Select products from a table and enter quantities.
- Add items to a cart.
- Enter customer details (name and contact).
- Generate a detailed bill.
- Save bills as a text file on your computer.

---

## Technologies Used
- Programming Language: **Python** 
- GUI Framework: **Tkinter** 
- Data Storage: **SQLite**

## Usage

- Open the application to access the **Dashboard**.  
- Navigate through the tabs to manage **employees, products, suppliers, categories, and sales**.  
- Use the **Billing Page** to create and save customer bills.
